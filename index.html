Browse and query the morphologically annotated corpus: every token tagged for part of speech, case, number, tense, mood, voice, gender, and degree.

Morphology tab

This tab is driven entirely by an in-browser SQLite database built from the corpus (table morphology). There are two ways to explore it:

  • Quick filter — pick a part of speech, number, and/or case from the drop-downs. MOPSOS writes the matching SQL for you, runs it, and shows the results 50 at a time (use the pager, or “Show all on one page”).
  • SQL console — write read-only SQL yourself (SELECT / WITH / PRAGMA). Results here are not paginated — your LIMIT is respected as written.

Reserved-word columns are quoted, e.g. WHERE "case" = 'g'.
